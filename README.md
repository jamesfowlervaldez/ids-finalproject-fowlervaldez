# Host-Based Intrusion Detection System (IDS)

## Project Overview
This project was developed as part of CIS 3370 - Intrusion Detection Systems at the University of the Incarnate Word.

The goal was to design and implement a host-based intrusion detection system capable of monitoring system activity, analyzing logs, and identifying potentially malicious behavior.

## Technologies Used
- Linux
- Shell Scripting
- Log Analysis
- Intrusion Detection Concepts

## Key Responsibilities
- Developed monitoring scripts
- Collected and analyzed system logs
- Configured IDS settings
- Generated reports and documentation
- Tested detection capabilities
